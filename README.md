## About

A NextJS + TypeScript application for generating random runs from a starting waypoint and a given distance.

Started from a rough single page HTML mockup generated by Haiku 4.5, and refined from there.

## Functionality

- Set a start location manually or use browser geolocation
- Choose a target distance and route style (loop/out-and-back)
- Generate a suggested running route on the map
- View useful route stats (such as distance and related metrics)
- Regenerate routes to explore alternatives from the same area

Routing is powered by **OSRM** (Open Source Routing Machine):  
https://project-osrm.org/
